## README Opening Copy

`Alfred-Dev-Tools is a growing collection of everyday developer utilities, packaged as a fast Alfred workflow. It helps you generate hashes, convert timestamps, format JSON, encode and decode text, inspect JWTs, generate UUIDs, and more without leaving your keyboard.`

## Release Copy

`Alfred-Dev-Tools is now available as an open-source workflow for macOS developers. It already includes password generation, MD5/SHA hashing, Base64 encode/decode, JSON formatting, URL encode/decode, UUID generation, timestamp conversion, text statistics, naming-case conversion, HTML escaping, JWT decoding, and timers. More developer utilities will keep landing over time.`

## Positioning

这个项目最适合这样介绍：

- 不是单一密码工具，而是开发者效率工具箱
- 不是大型 GUI App，而是轻量、快速、键盘优先
- 不是一次性脚本集合，而是持续扩展的模块化 Workflow
